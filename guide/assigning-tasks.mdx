title: Assigning Tasks
description: Create and assign tasks to team members using drag-and-drop assignment and visual color indicators
---

Create tasks, assign them to team members, and manage workload distribution using Gantt Maker's intuitive visual assignment system.

## Creating a New Task

<Steps>
  <Step title="Add Task">
    Click the add task button or option in your Gantt chart interface to create a new task entry.
  </Step>
  
  <Step title="Enter Task Name">
    Provide a clear, descriptive name for the task that team members will easily understand.
  </Step>
  
  <Step title="Add Task Description">
    Include detailed information about the task, including objectives, deliverables, and any important notes. Clear descriptions help team members understand exactly what needs to be done.
  </Step>
  
  <Step title="Set Task Duration">
    Define how long the task will take to complete. You can set duration in:
    - **Date mode**: Specify days or weeks
    - **Hour mode**: Define precise hours for time-sensitive work
  </Step>
</Steps>

## Assigning Tasks to Team Members

Gantt Maker provides flexible options for assigning tasks:

### Drag and Drop Assignment

The most intuitive way to assign tasks:

<Steps>
  <Step title="Select the Task">
    Click and hold on the task you want to assign.
  </Step>
  
  <Step title="Drag to Team Member">
    Drag the task to the desired team member in the chart. The intuitive drag-and-drop interface makes reassignment quick and easy.
  </Step>
  
  <Step title="Drop to Assign">
    Release to assign the task. The task immediately takes on the team member's unique color.
  </Step>
</Steps>

### Visual Assignment with Colors

Once assigned, tasks display visual indicators:

- **Team member color**: The task bar shows the assigned member's unique color
- **Instant recognition**: Quickly identify who owns each task
- **Workload patterns**: See distribution of work across your team at a glance

## Setting Task Timeline

Position tasks on your timeline with precision:

<Steps>
  <Step title="Choose Start Point">
    Click or drag the task to the desired start date or time on the Gantt chart.
  </Step>
  
  <Step title="Adjust Duration">
    Resize the task bar by dragging its edges to set the appropriate duration.
  </Step>
  
  <Step title="Verify Placement">
    Check that the task fits within the timeline and doesn't create conflicts with other assignments.
  </Step>
</Steps>

## Reassigning Tasks

Move tasks between team members effortlessly:

<Steps>
  <Step title="Select Assigned Task">
    Click on the task you want to reassign.
  </Step>
  
  <Step title="Drag to New Member">
    Use drag and drop to move the task to a different team member's row.
  </Step>
  
  <Step title="Confirm Reassignment">
    Drop the task to complete the reassignment. The task color updates instantly to reflect the new assignment.
  </Step>
</Steps>

## Editing Task Details

Modify task information in real-time:

<Steps>
  <Step title="Open Task Editor">
    Click on the task to access its details and editing options.
  </Step>
  
  <Step title="Update Information">
    Edit the task name, description, duration, or any other details as needed.
  </Step>
  
  <Step title="Save Changes">
    Changes apply immediately and are automatically saved to your local storage.
  </Step>
</Steps>

<Tip>
The tool automatically detects over-assignments and conflicts, helping you identify when team members have overlapping tasks or excessive workload.
</Tip>

## Best Practices

- **Clear descriptions**: Write comprehensive task descriptions to reduce ambiguity
- **Realistic durations**: Set achievable timeframes based on team capacity
- **Balance workload**: Use the visual color distribution to ensure no team member is overloaded
- **Regular updates**: Keep task information current as requirements evolve
- **Sequential planning**: Consider task dependencies when assigning start dates
