# Frequently Asked Questions

Find answers to the most common questions about Gantt Maker.

## Getting Started

<Accordion title="What are the system requirements for Gantt Maker?">
  Gantt Maker is a web-based application that runs on any modern browser. You need:
  - A modern web browser (Chrome, Firefox, Safari, Edge)
  - No internet connection required after initial load
  - No installation or dependencies needed
  - Works on desktop, tablet, and mobile devices
</Accordion>

<Accordion title="Which browsers are compatible with Gantt Maker?">
  Gantt Maker works on all modern browsers including:
  - Google Chrome (recommended)
  - Mozilla Firefox
  - Safari
  - Microsoft Edge
  - Opera
  
  The application uses modern web standards (HTML5, CSS3, vanilla JavaScript) for optimal performance.
</Accordion>

## Data and Storage

<Accordion title="Where is my data stored?">
  All your data is stored locally on your device using LocalStorage. This means:
  - Your data never leaves your computer
  - No cloud storage or external servers
  - Complete privacy and control
  - Auto-save functionality keeps your changes secure
  - Data persists between sessions on the same browser
</Accordion>

<Accordion title="Can I use Gantt Maker offline?">
  Yes! Gantt Maker is designed to work completely offline. Once loaded in your browser, you can:
  - Create and edit projects without internet connection
  - All features work offline
  - Data is saved locally automatically
  - Perfect for working on the go or in restricted environments
</Accordion>

<Accordion title="How do I backup my data?">
  You can backup your projects by:
  - Using the native export functionality to save projects as JSON files
  - Exporting multiple project versions for different scenarios
  - Storing exported files in your preferred backup location
  - Importing JSON files to restore your projects
  
  Regular exports are recommended to maintain backup copies of your important projects.
</Accordion>

<Accordion title="Can I export my Gantt charts?">
  Yes, Gantt Maker supports native export functionality:
  - Export projects as JSON files
  - Share projects easily with team members
  - Import previously exported projects
  - Create multiple versions of the same project
  - Full interoperability with your existing tools
</Accordion>

## Collaboration and Teams

<Accordion title="Does Gantt Maker support multi-user collaboration?">
  Gantt Maker is designed as a single-user application with local storage. For team collaboration:
  - Export your project as a JSON file
  - Share the file with team members via email, cloud storage, or file sharing
  - Team members can import and edit the project
  - Exchange updated versions as needed
  
  This approach gives you complete control over who accesses your project data.
</Accordion>

<Accordion title="Can I manage multiple team members in a project?">
  Absolutely! Gantt Maker includes powerful team management features:
  - Add unlimited team members to your project
  - Assign unique colors to each team member for visual clarity
  - Define roles and specialties for each member
  - Use drag-and-drop to reassign tasks
  - Visual color-coding helps identify workload and conflicts instantly
</Accordion>

## Features and Functionality

<Accordion title="What's the difference between date mode and hour mode?">
  Gantt Maker offers two timeline modes:
  
  **Date Mode**: Traditional planning by days and weeks
  - Best for long-term projects
  - Week and month views
  - Ideal for milestone planning
  
  **Hour Mode**: Precise planning down to the hour
  - Perfect for intensive short-term projects
  - Detailed time tracking
  - Great for sprint planning and task scheduling
  
  You can switch between modes instantly with a single click.
</Accordion>

<Accordion title="Can I exclude weekends and holidays from my timeline?">
  Yes! Gantt Maker includes intelligent scheduling features:
  - Exclude weekends automatically
  - Define specific dates to exclude (holidays, company closures)
  - Set custom non-working hour blocks
  - Adapt the timeline to your actual workflow
  - Ensures realistic project timelines
</Accordion>

<Accordion title="How do I track task progress?">
  Task progress tracking is simple and interactive:
  - Click on any task to cycle through states
  - States: Not Started → In Progress (WIP) → Completed
  - Visual indicators show status at a glance
  - Real-time updates to your timeline
  - Edit task details directly in the interface
</Accordion>

## Privacy and Security

<Accordion title="Does Gantt Maker track my usage or collect data?">
  No. Gantt Maker has zero tracking and data collection:
  - No analytics or telemetry
  - No user accounts or registration
  - No data sent to external servers
  - Your project data remains completely private
  - No cookies or tracking technologies
</Accordion>

<Accordion title="Is there a cost to use Gantt Maker?">
  Gantt Maker is completely free:
  - No subscription fees
  - No licensing costs
  - No feature limitations
  - No hidden charges
  - Full functionality available to all users
</Accordion>
